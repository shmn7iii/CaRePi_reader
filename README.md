# CaRePi_reader

Reader of CaRePi.

> CaRePi is **Ca**rd **Re**ader de **Pi** tte suru yatu

## setup

### 1. install requirements

```bash
$ pip3 install -r requirements.txt
```

### 2. setup nfc

```
$ sudo python3 -m nfc

# if error
$ sudo modprobe -r port100
```
